Create a Hugo Blog Post
Mar 20, 2024
  • If you haven’t done so already, setup Hugo with github pages
  • Create a new post markdown file:
➜ hugo new content posts/hugo-posts.md
  • In order for the new post to be displayed in blog site, delete the draft: true line from the frontmatter yaml at the top of the markdown file:
---
title: "Create a Hugo Blog Post"
date: 2024-03-20T17:42:24+09:00
draft: true
---
  • Write your blog (you’re on your own here)
  • Create the static html resources for the new post, using the holy theme:
➜ hugo -t holy
  • Update the remote github pages repo:
➜  cd public
➜  git add .
➜  git commit -m "Add blog about creating a new hugo blog post"
